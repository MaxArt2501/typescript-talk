Getting started

Try everything on the spot!

1. From npm

$ npm i -g typescript
$ tsc my-module.ts --out my-module.js

2. ts-node

$ npm i -g ts-node
Use it just like node
$ ts-node my-module.ts

3. On the web

typescript-play.js.org

4. In your IDE

tsconfig.json

{
  "include": [ "src/**/*.ts" ],
  "exclude": [ "src/**/*.spec.ts" ],
  "compilerOptions": {
    "target": "ES2015",
    "outDir": "dist",
    "strictNullChecks": true, ... } }
